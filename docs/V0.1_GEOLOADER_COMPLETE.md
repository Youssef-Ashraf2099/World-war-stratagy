## 🏗️ Architecture

### Class Structure

```
Engine::Subsystems::Geography::GeoLoader
├── loadNations(filePath) → bool
├── getNations() → vector<GeoNationData>
├── getNationById(id) → GeoNationData*
├── getNationByCode(code) → GeoNationData*
├── getNationByName(name) → GeoNationData*
└── getNationCount() → size_t
```

### Data Flow

```
Natural Earth Shapefile (.dbf)
    ↓ (extract_nations.py)
JSON File (nations.json)
    ↓ (GeoLoader::loadNations)
GeoNationData (in-memory)
    ↓ (main.cpp conversion)
Engine::Core::Types::Nation (game objects)
    ↓
WorldState
```
