## Overview

The Alliance System manages diplomatic unions between nations. Alliances provide cohesion mechanics, doctrine-based obligations, and strategic benefits through reduced threat perception during wars.

---

## Core Concepts

### Alliance Lifecycle

```
Formation → Active → Decay → Dissolution
   (T0)    (T0-T∞)  (T1-T∞)    (cohesion<15)
```

**States:**
1. **Formation** - Alliance created with initial nations
2. **Active** - Executing doctrines, members cooperate
3. **Decay** - Cohesion diminishes each tick
4. **Dissolution** - Automatic breakup when cohesion < 15 or members < 2

### Cohesion Mechanics

Cohesion represents alliance stability (0-100):
- **High cohesion** (80-100): Strong alliance bonds, fast obligation enforcement
- **Medium cohesion** (40-79): Stable operation, normal mechanics
- **Low cohesion** (15-40): Strained relationships, slow to respond
- **Critical** (0-14): Alliance dissolved, removed from simulation

**Decay Formula:**
```
each tick: new_cohesion = max(0, cohesion - cohesion_decay_rate)
```

**Example:**
- Starting cohesion: 80.0
- Decay rate: 2.0 per tick
- After 5 ticks: 80.0 - (2.0 × 5) = 70.0
- After 40 ticks: 0.0 (dissolved)

## Alliance Doctrines

Five doctrine types define how alliances operate:

### 1. Defensive Agreement
**Purpose:** Mutual defense pact  
**Mechanic:** Members auto-join wars defending other members  
**Cohesion Cost:** 1.5 per tick  
**Threat Reduction:** 0.30 (moderate)

**Effect:**
```rust
If member_a is attacked by aggressor_x:
  → All other members immediately join defense
  → Aggressor faces opposition from entire alliance
  → Reduces effective military threat by 30%
```

**Use Case:** NATO-style collective defense

---

### 2. Offensive Pact
**Purpose:** Coordinated military expansion  
**Mechanic:** Members coordinate attacks on shared enemies  
**Cohesion Cost:** 2.0 per tick (high cost)  
**Threat Reduction:** 0.15 (low)

**Effect:**
```rust
If member_a declares war on enemy_x:
  → Members receive bonuses for attacking same enemy
  → Coordinated battles possible
  → Multiplier effect on combat resolution
```

**Use Case:** Historical: Axis powers, modern: military coalitions

---

### 3. Economic Bloc
**Purpose:** Trade and economic cooperation  
**Mechanic:** Reduced trade friction, tariff benefits  
**Cohesion Cost:** 0.8 per tick (low cost)  
**Threat Reduction:** 0.25 (moderate)

**Effect:**
```rust
Members enjoy:
  → +15% trade efficiency between bloc members
  → Tariff protection from non-members
  → Shared resource purchasing power
```

**Use Case:** EU, ASEAN, MERCOSUR

---

### 4. Research Consortium
**Purpose:** Scientific and technological collaboration  
**Mechanic:** Shared research accelerates tech advancement  
**Cohesion Cost:** 0.5 per tick (lowest cost)  
**Threat Reduction:** 0.20 (low)

**Effect:**
```rust
Members:
  → Pool 15% of research budget into shared projects
  → Distributed advances across all members
  → Reduced time to key technologies
```

**Use Case:** CERN, scientific partnerships

---

### 5. Balance of Power
**Purpose:** Regional stability through mutual restraint  
**Mechanic:** Members agree to limit regional arms races  
**Cohesion Cost:** 1.2 per tick  
**Threat Reduction:** 0.35 (highest)

**Effect:**
```rust
When member engaged in conflict:
  → Other members provide diplomatic pressure
  → War exhaustion increases 20% slower
  → De-escalation mechanics activate
```

**Use Case:** Concert of Europe, regional balancing

## Core Data Structure

```rust
pub struct Alliance {
    // Identity
    pub alliance_id: AllianceId,                // UUID
    pub alliance_name: String,                  // "NATO", "ASEAN", etc.
    pub founding_nation: NationId,              // Original founder
    
    // Membership
    pub members: Vec<NationId>,                 // All member nation IDs
    
    // Status
    pub cohesion: f64,                          // 0-100, decays each tick
    pub doctrine: AllianceDoctrine,             // Five types above
    pub founded_tick: Tick,                     // When created
    
    // Mechanics
    pub threat_reduction: f64,                  // 0.15-0.50 multiplier
    pub cohesion_decay_rate: f64,               // 0.5-2.5 points/tick
}

impl Alliance {
    pub fn member_count(&self) -> usize
    pub fn is_dissolved(&self) -> bool          // cohesion < 15 || members < 2
    pub fn decay_cohesion(&mut self)            // Decays by decay_rate
    pub fn boost_cohesion(&mut self, amount)    // +amount (clamped to 100)
    pub fn add_member(&mut self, nation: NationId)
    pub fn remove_member(&mut self, nation: NationId)
}
```

## Alliance Phase Execution

**When:** Tick phase 8 (after occupation, before diplomacy)  
**Duration:** O(n) where n = number of alliances  
**Deterministic:** Yes (same orders every run)

### Step 1: Cohesion Decay
```rust
for each alliance:
    alliance.cohesion -= alliance.cohesion_decay_rate
    alliance.cohesion = max(0.0, alliance.cohesion)
```

**Result:** Alliance bonds gradually weaken over time

### Step 2: Dissolution Check
```rust
for each alliance:
    if alliance.is_dissolved():
        // Remove alliance entity (effectively deletes it)
        // Members revert to independent status
```

**Result:** Dead alliances cleared from simulation

### Step 3: Obligation Enforcement
```rust
for each DefensiveAgreement alliance:
    for each ongoing war:
        if defender in alliance.members:
            for each other member:
                add_member_to_war_support(other_member, defender)
```

**Result:** Defensive agreements auto-trigger

## Integration with Other Systems

### Warfare Integration

**DefensiveAgreement Obligations:**
```
When war declared against alliance member:
  1. WarfarePhase registers war declaration (Phase 2)
  2. ... combat happens (Phases 6-7)
  3. AlliancePhase checks all defensive agreements (Phase 8)
  4. Other members auto-added to WarState.at_war_with
  5. Next tick: AI & combat calculations include all allies
```

### Stability Integration

**Cohesion Effects on Legitimacy:**
```
Alliance members with high cohesion:
  → +2.5 legitimacy bonus (sense of security)
  → -10% protest chance (strong external bonds)

Alliance members with low cohesion:
  → No bonus (unstable arrangement)
  → Potential dissolution triggers unrest
```

### Diplomacy Integration

**Reputation & Alliances:**
```
Member meets same ally:
  → Reputation boost (shared interests)
  
Member meets enemy of ally:
  → Reputation penalty (conflicting loyalty)
  
Alliance dissolution:
  → -15 reputation to all former members (betrayal)
```

## Game Rules

### Membership Rules
```
✓ Nations can join multiple alliances
✗ Nations cannot join conflicting doctrines
  (e.g., OffensivePact against their defensive ally)
✓ Founding nation doesn't get special status
✗ Cannot leave alliance voluntarily
  (must wait for dissolution due to low cohesion)
```

### Alliance Formation Rules
```
Nations can propose alliances through AI:
  → Score calculated by DiplomacyPhase
  → Threshold: score > 0.6
  → Founder gets founding_nation status

Manual alliance creation (save files):
  → Loaded from StateSnapshot.alliances
  → Restored with full cohesion/doctrine
```

### War Rules
```
DefensiveAgreement members defend:
  → If ANY member attacked
  → All others auto-join (immediate)
  → Applied even if attacker is ally (complex!)

OffensivePact members attack:
  → Can coordinate strikes
  → No auto-join (must declare separately)
  → Bonus to joint operations
```

## Examples

### Example 1: Create Alliance Programmatically

```rust
use crate::core::types::{Alliance, AllianceId, AllianceDoctrine, NationId};

let nation_a = NationId::new();
let nation_b = NationId::new();

let alliance = Alliance {
    alliance_id: AllianceId::new(),
    alliance_name: "Test Alliance".to_string(),
    founding_nation: nation_a,
    members: vec![nation_a, nation_b],
    cohesion: 95.0,              // Very stable
    doctrine: AllianceDoctrine::DefensiveAgreement,
    founded_tick: 0,
    threat_reduction: 0.30,
    cohesion_decay_rate: 1.5,
};

world.spawn(alliance);
```

### Example 2: Monitor Alliance Decay

```rust
let mut query = world.query::<&Alliance>();
for alliance in query.iter(&world) {
    println!("Alliance: {}", alliance.alliance_name);
    println!("  Cohesion: {:.1}", alliance.cohesion);
    println!("  Members: {}", alliance.member_count());
    println!("  Decaying at: {:.1}/tick", alliance.cohesion_decay_rate);
    println!("  Will dissolve in: {:.0} ticks", 
             alliance.cohesion / alliance.cohesion_decay_rate);
}
```

### Example 3: Handle Dissolution

```rust
let mut to_remove = Vec::new();

let mut query = world.query::<&Alliance>();
for (entity, alliance) in query.iter_with(world) {
    if alliance.is_dissolved() {
        to_remove.push(entity);
        println!("Alliance {} dissolved!", alliance.alliance_name);
    }
}

for entity in to_remove {
    world.despawn(entity);
}
```

### Example 4: Query Member Alliances

```rust
let target_nation = NationId::new();

let mut query = world.query::<&Alliance>();
for alliance in query.iter(&world) {
    if alliance.members.contains(&target_nation) {
        println!("Nation is member of: {}", alliance.alliance_name);
        println!("  Members: {}", alliance.member_count());
        println!("  Doctrine: {:?}", alliance.doctrine);
    }
}
```

## Common Pitfalls

### ❌ Pitfall 1: Forgetting Dissolution
```rust
// Wrong: Assumes alliance exists every tick
let cohesion = alliance.cohesion;  // Might not exist!

// Right: Check before accessing
if !alliance.is_dissolved() {
    let cohesion = alliance.cohesion;
}
```

### ❌ Pitfall 2: Assuming Member Status Stable
```rust
// Wrong: Member list doesn't change mid-tick
let members = &alliance.members;  // Could invalidate

// Right: Query fresh each phase
for (entity, mut alliance) in query.iter_mut(world) {
    for member in &alliance.members {
        // Use member_id
    }
}
```

### ❌ Pitfall 3: Ignoring Doctrine Effects
```rust
// Wrong: All alliances behave the same
if member_attacked {
    // Auto-join war for everyone
}

// Right: Check doctrine
if member_attacked && doctrine == DefensiveAgreement {
    // Only these auto-join
}
```
