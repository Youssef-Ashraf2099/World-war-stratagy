## 🎯 WHAT'S NEXT: STRATEGIC RECOMMENDATIONS

### Phase 5: V0.5 — Legitimacy & Internal Pressure (RECOMMENDED NOW)

**Why it's critical:**

- Alliances create _external_ commitments
- Legitimacy adds _internal_ constraints
- Together = realistic geopolitical cycles (rise/fall patterns)
- Without it: Empires can sustain unlimited overextension

**Effort:** 3-4 weeks

**What to build:**

1. **Legitimacy scalar** (0-100) for each nation
2. **War exhaustion** mechanic (-0.5 to -1.5/tick per active war)
3. **Resource stress** penalties (-0.2 to -0.5/tick per deficit type)
4. **Alliance strain** (obligations create legitimacy loss if refused)
5. **Collapse condition** (legitimacy = 0 → nation removal or failed state)
6. **Recovery mechanism** (peace provides +0.3/tick recovery)

**Single implementation:** One new `LegitimacyPhase` (insert at phase 11, after Demographics)

**Impact:**

- Adds realistic constraints to empire building
- Enables historical-style rise/fall cycles
- Creates strategic depth (trade-off: overextend vs legitimacy)
- TOTAL tests: 110+ (adding ~20 new tests)

**Key formula:**

```
legitimacy_change =
  (-0.5 to -1.5 × active_wars) +
  (-0.2 to -0.5 if deficits) +
  (-0.1 to -0.3 if alliance obligations) +
  (+0.3 base recovery if peace)

new_legitimacy = clamp(old + change, 0, 100)

if new == 0: nation_collapses()
```

**Recommended timeline:**

- Week 1: Core legitimacy system + unit tests
- Week 2: Integration with Economy/Warfare/Alliance/Diplomacy
- Week 3: Validation + documentation
