## Quick Start

### Migrate from V0.35

**Step 1: Update Pipeline Constructor**

```rust
// OLD (V0.35)
let mut pipeline = TickPipeline::new_v0_35();

// NEW (V0.4)
let mut pipeline = TickPipeline::new_v0_4();
```

That's it! Alliances and diplomacy are now in the simulation.

### Minimal Example

```rust
use alalamien_engine::core::{world::WorldState, tick::TickPipeline};

fn main() {
    // Create world
    let mut world = WorldState::new(42);
    
    // Create nations
    let nation_a = world.spawn_nation("Nation A".to_string(), [255, 0, 0], true);
    let nation_b = world.spawn_nation("Nation B".to_string(), [0, 0, 255], false);
    
    // Run V0.4 simulation
    let mut pipeline = TickPipeline::new_v0_4();
    
    for _tick in 0..100 {
        pipeline.execute(&mut world);
    }
    
    // Alliances and diplomacy evolve automatically!
}
```

### Monitoring Alliances

```rust
// Check all alliances
let mut query = world.query::<&Alliance>();
for alliance in query.iter(&world) {
    println!("Alliance: {}", alliance.alliance_name);
    println!("  Cohesion: {:.1}", alliance.cohesion);
```

### Adding/Removing Members

```rust
let mut query = world.query::<&mut Alliance>();
for mut alliance in query.iter_mut(&world) {
    if alliance.alliance_name == "Target Alliance" {
        alliance.add_member(new_nation_id);
        // or
        alliance.remove_member(old_nation_id);
    }
}
```

### Boosting Cohesion

```rust
// Increase cohesion (clamped to 100)
alliance.boost_cohesion(10.0);

// Decay cohesion (happens automatically in AlliancePhase)
alliance.decay_cohesion();
```

### Handling Dissolution

```rust
let mut to_remove = Vec::new();

let mut query = world.query::<&Alliance>();
for (entity, alliance) in query.iter_with(&world) {
    if alliance.is_dissolved() {
        println!("Alliance {} dissolved!", alliance.alliance_name);
        to_remove.push(entity);
    }
}

for entity in to_remove {
    world.despawn(entity);
}
```

```rust
        println!("Relation with: {:?}", relation.nation_a);
        println!("  Reputation: {:.1}", relation.reputation);
        println!("  Score: {:.2}", relation.alliance_score());
    }
}
```

### Modifying Reputation

```rust
let mut query = world.query::<&mut DiplomaticRelation>();
for mut relation in query.iter_mut(&world) {

    relation.modify_reputation(-10.0);
```

### Checking Relationship Status

## AI Integration: Alliance Proposals

### How It Works

1. **Each AI decision tick:**
   - AI evaluates all potential allies
   - Scores via: `(0.3×trade) + (0.4×|threat|) + (0.3×rep)`
   - Proposes to scored > 0.6

2. **On proposal:**
   - Updates diplomatic relation
   - Reputation boosted +5.0
   - AI can then create alliance

### AI-Generated Proposals

The AI automatically proposes alliances when appropriate:

```rust
// In AIDecisionPhase::execute()
if let Some(AdvancedDecision::ProposeAlliance { candidate, alliance_id }) = 
    choose_decision(&mut world, snapshot, personality, &snapshots, nation_id) 
{
    // AI found a good alliance match and proposed
    // DiplomaticRelation updated with +5.0 reputation
}
```

### Manual Alliance Proposals

For custom logic, you can manually trigger proposals:

```rust
// Find best potential ally
let candidates = find_alliance_candidates(&mut world, nation_a);

for candidate in candidates {
    let score = DiplomacyPhase::alliance_proposal_score(
        &mut world, 
        nation_a, 
        candidate
    );
    
    if score > 0.6 {
        // Boost reputation as if proposed
        if let Some(mut relation) = find_relation(&mut world, nation_a, candidate) {
            relation.modify_reputation(5.0);
            
            // Then create alliance
            let alliance = Alliance {
                alliance_id: AllianceId::new(),
                alliance_name: "Custom Alliance".to_string(),
                founding_nation: nation_a,
                members: vec![nation_a, candidate],
                cohesion: 80.0,
                doctrine: AllianceDoctrine::DefensiveAgreement,
                founded_tick: world.current_tick(),
                threat_reduction: 0.30,
                cohesion_decay_rate: 1.5,
            };
            
            world.spawn(alliance);
        }
    }
}
```

### Customizing AI Behavior

Extend the AI with your own alliance logic:

```rust
enum CustomDecision {
    ProposeAlliance { candidate: NationId },
    ForcefulAlliance { target: NationId }, // Custom: forced alliance
    BreakAlliance { alliance_id: AllianceId }, // Custom: coalition breaking
}

// Then in your execute()
match decision {
    CustomDecision::ForcefulAlliance { target } => {
        // Military pressure to join alliance
        let mut relation = find_relation(&mut world, self_id, target);
        relation.modify_reputation(-20.0); // Threaten
        
        let alliance = create_forced_alliance(self_id, target);
        world.spawn(alliance);
    },
    _ => { /* ... */ }
}
```

## Tick Pipeline

### V0.4 Execution Order

```
Tick begins
  ↓
1. AdvancedAIDecisionPhase
   ├→ Proposes alliances (score > 0.6)
   └→ Updated DiplomaticRelation (reputation +5)
  ↓
2. WarfarePhase
  ↓
3-7. Economy, Trade, Logistics, Combat, Occupation
  ↓
8. AlliancePhase ← NEW
   ├→ Decays cohesion
   ├→ Checks dissolutions
   └→ Enforces obligations (DefensiveAgreement)
  ↓
9. DiplomacyPhase ← NEW
   ├→ Updates relations from wars
   ├→ Modifies threat alignment
   └→ Decays reputation
  ↓
10-11. Stability, Demographics
  ↓
Tick ends
```

### Custom Phase Order

If you need different order, create custom pipeline:

```rust
let mut custom_pipeline = TickPipeline::new_v0_4();

// Remove & re-add phases in different order
custom_pipeline.clear(); // Not available yet, would need API
custom_pipeline.add_phase(Box::new(DiplomacyPhase::new()));
custom_pipeline.add_phase(Box::new(AlliancePhase::new()));
// ... etc
```

### Monitoring Phase Execution

```rust
// Before/after snapshots
let hash_before = world.state_hash();
pipeline.execute(&mut world);
let hash_after = world.state_hash();

if hash_before != hash_after {
    println!("World state changed this tick");
}
```

## Testing Integration

### Basic Simulation Test

```rust
#[test]
fn test_alliance_diplomacy_interaction() {
    let mut world = WorldState::new(42);
    let mut pipeline = TickPipeline::new_v0_4();
    
    // Setup
    let nation_a = world.spawn_nation("A".to_string(), [255, 0, 0], false);
    let nation_b = world.spawn_nation("B".to_string(), [0, 0, 255], false);
    
    let nation_a_id = world.world.get::<Nation>(nation_a).unwrap().id;
    let nation_b_id = world.world.get::<Nation>(nation_b).unwrap().id;
    
    // Create alliance
    let alliance = Alliance {
        alliance_id: AllianceId::new(),
        alliance_name: "Test".to_string(),
        founding_nation: nation_a_id,
        members: vec![nation_a_id, nation_b_id],
        cohesion: 50.0,
        doctrine: AllianceDoctrine::DefensiveAgreement,
        founded_tick: 0,
        threat_reduction: 0.30,
        cohesion_decay_rate: 2.0,
    };
    world.world.spawn(alliance);
    
    // Run 30 ticks
    pipeline.execute_many(&mut world, 30);
    
    // Verify coalition still exists but decayed
    let mut found = false;
    let mut query = world.world.query::<&Alliance>();
    for alliance in query.iter(&world.world) {
        if alliance.alliance_name == "Test" {
            found = true;
            assert!(alliance.cohesion < 50.0); // Decayed
            assert!(!alliance.is_dissolved()); // Still active
        }
    }
    assert!(found, "Alliance should still exist");
}
```

### Determinism Test

```rust

fn test_v0_4_determinism() {
    // Run 1
    let mut world1 = WorldState::new(1234);
    let mut pipeline1 = TickPipeline::new_v0_4();
    setup_world(&mut world1);
    let hash1_before = world1.state_hash();
```

## Common Workflows

### Workflow 1: Check Alliance Status

**Goal:** Get overview of alliances affecting a nation

```rust
fn get_nation_alliance_status(world: &World, nation_id: NationId) -> String {
    let mut status = String::new();
    
    let mut query = world.query::<&Alliance>();
    for alliance in query.iter(world) {
        if alliance.members.contains(&nation_id) {
            status.push_str(&format!(
                "{} (cohesion: {:.1}, {} members)\n",
                alliance.alliance_name,
                alliance.cohesion,
                alliance.member_count()
            ));
        }
    }
    
    if status.is_empty() {
        status = "No alliances".to_string();
    }
    
    status
}
```

### Workflow 2: Find Potential Allies

**Goal:** Identify good alliance candidates

```rust
fn find_alliance_candidates(
    world: &mut World, 
    nation: NationId,
    threshold: f64
) -> Vec<(NationId, f64)> {
    let mut candidates = Vec::new();
    
    let nations: Vec<NationId> = {
        let mut query = world.query::<(&Nation, &WarState)>();
        query
            .iter(world)
            .map(|(n, _)| n.id)
            .collect()
    };
    
    for other in nations {
        if other == nation { continue; }
        if is_enemy(world, nation, other) { continue; }
        
        let score = DiplomacyPhase::alliance_proposal_score(
            world, nation, other
        );
        
        if score > threshold {
            candidates.push((other, score));
        }
    }
    
    candidates.sort_by(|a, b| b.1.partial_cmp(&a.1).unwrap());
    candidates
}
```

### Workflow 3: Track Diplomatic Changes

**Goal:** Log all reputation changes this tick

```rust
fn log_diplomatic_changes(
    world: &World,
    previous_relations: &[RelationSnapshot]
) {
    let mut query = world.query::<&DiplomaticRelation>();
    for current in query.iter(world) {
        if let Some(previous) = find_relation_snapshot(
            previous_relations, 
            current.nation_a, 
            current.nation_b
        ) {
            let rep_change = current.reputation - previous.reputation;
            if rep_change.abs() > 0.1 {
                println!(
                    "Reputation change {:?} <-> {:?}: {:+.1}",
                    current.nation_a, current.nation_b, rep_change
                );
            }
        }
    }
}
```

## Performance Considerations

### Overhead of V0.4

```
AlliancePhase:  O(n) where n = alliances (typically << 100)
  Cohesion:     ~0.1ms per 100 alliances
  Dissolution:  ~0.05ms per 100 alliances
  
DiplomacyPhase: O(n²) where n = nations
  War updates:  ~0.5ms per 100 nations
  Threat calc:  ~5-10ms for 200 nations (largest cost)
  Reputation:   ~0.2ms per 1000 relations
```

**For typical 50-200 nation games:**
- AlliancePhase: < 1ms overhead
- DiplomacyPhase: 1-10ms overhead (threat alignment quadratic)
- Total: < 15ms per tick

### Optimization Tips

1. **Limit nation count** - Threat alignment is O(n²)
2. **Batch queries** - Query once, iterate multiple times
3. **Cache scores** - Diplomacy score doesn't change mid-tick
4. **Lazy updates** - Only update relations that matter

## Troubleshooting

### Problem: Alliances Never Propose

**Causes:**
1. No diplomatic relations exist
2. Scores all < 0.6
3. AI phase not running

**Solution:**
```rust
// Check if AI is enabled
let advanced_ai = AdvancedAIDecisionPhase::new();

// Manually calculate scores
let score = DiplomacyPhase::alliance_proposal_score(&mut world, nation_a, nation_b);
assert!(score > 0.0, "Score too low");
```

### Problem: Alliances Dissolve Too Quickly

**Cause:** cohesion_decay_rate too high

**Solution:**
```rust
// Reduce decay
alliance.cohesion_decay_rate = 0.5;  // Was 2.5

// Or boost cohesion
alliance.boost_cohesion(25.0);
```

### Problem: Determinism Changes Between Runs

**Cause:** Likely floating-point rounding in threat calculations

**Solution:**
- Use same seed
- Run same number of ticks
- Don't modify state mid-tick
