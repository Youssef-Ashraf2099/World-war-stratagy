## What's New in V0.4

### 1. Alliance System

Nations can now join diplomatic alliances with:

- **Cohesion tracking** (decays each tick)
- **Doctrine-based mechanics** (DefensiveAgreement auto-joins wars)
- **Obligation enforcement** (allies defend allies)
- **Dissolution detection** (< 15 cohesion triggers breakup)

**27 Alliance Types:**

- 5 Military alliances (NATO-style)
- 5 Economic blocs (EU-style)
- 5 Scientific consortiums (research cooperation)
- 5 Cultural unions (soft power)
- 2 Regional agreements

### 2. Diplomacy Subsystem

Track and evolve relationships between nations:

- **Reputation** (-100 to +100)
- **Trade dependency** (0-1 influence)
- **Threat alignment** (-1 to +1)
- **War history** (last_war timestamp)
- **Allied status** (allied_since timestamp)

### 3. AI Enhancement

Advanced AI now **autonomously proposes alliances**:

- Evaluates all potential allies each tick
- Scores via diplomacy formula: `0.3×trade + 0.4×threat + 0.3×reputation`
- Proposes when score > 0.6 threshold
- Avoids proposing to enemies
- Updates reputation (+5.0) on successful proposal

### 4. REST API

Three new endpoints expose alliance/diplomacy data:

```
GET /alliances                          → All alliances + members
GET /alliances/:nation_id               → Nation's memberships
GET /diplomacy/:nation_a/:nation_b      → Bilateral relations
```

### 5. Tick Pipeline Integration

V0.4 tick pipeline now executes 11 phases:

```
1. Advanced AI Decision (alliance proposals)
2. Warfare
3. Economy
4. Trade
5. Logistics
6. Combat
7. Occupation
8. Alliance Phase ← NEW (cohesion decay, dissolution)
9. Diplomacy Phase ← NEW (relation updates, threat alignment)
10. Stability
11. Demographics
```

## Core Formulas

### Alliance Proposal Score

```
score = (0.3 × trade_dependency)
      + (0.4 × |threat_alignment|)
      + (0.3 × (reputation + 100) / 200)

Proposal triggered when: score > 0.6
```

### Cohesion Decay

```
new_cohesion = max(0, current_cohesion - cohesion_decay_rate)
Alliance dissolved when: cohesion < 15.0 OR members < 2
```

### Threat Alignment Update

```
shared_enemies_boost = shared_enemy_count × 0.1
direct_conflict_penalty = (has_conflict ? 1.0 : 0.0) × 0.3
threat_delta = shared_enemies_boost - direct_conflict_penalty
```

### Reputation Decay

```
REPUTATION_DECAY = 0.5 per tick
new_reputation = reputation - sign(reputation) × REPUTATION_DECAY
(drifts towards 0.0 over time)
```

## Known Limitations

1. **Alliance dissolution is binary** - alliances dissolve immediately when conditions met, no gradual breakup process
2. **No alliance hierarchy** - all members have equal status (could support vassal states in V0.5)
3. **Trade dependency static** - not affected by actual trade volume (will be integrated with V0.5 trade routes)
4. **AI doesn't break alliances** - only proposes, never dissolves (will add in V0.5)
5. **No historical records** - diplomatic incidents aren't logged for player review (quality-of-life feature)
