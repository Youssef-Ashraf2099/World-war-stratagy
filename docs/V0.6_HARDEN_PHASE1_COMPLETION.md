### Determinism Validation

**Mechanism:**

```rust
// All randomness seeded from DeterministicRng(99999)
world.insert_resource(DeterministicRng::new(99999));

// Every subsystem uses deterministic RNG:
// - AI decisions (attack % calculations)
// - War outcomes (casualty rolls)
// - Event generation (stability events)
// - Economic models (resource variations)
// - Intervention mechanics (refugee population, protectorate chance)

// Result: Same seed → Same RNG sequence → Same outcomes
```

**Verification:**

```rust
// Each checkpoint records world state metrics
let checkpoint = (tick_count, world.nations().count(), world.provinces().count());

// All 3 runs must have identical checkpoint sequences
assert_eq!(checkpoints_1[i], checkpoints_2[i], "Run 1 vs Run 2");
assert_eq!(checkpoints_1[i], checkpoints_3[i], "Run 1 vs Run 3");
```
