## ⚠️ Known Limitations (Not Bugs)

1. **Event Effects Not Cascading Yet**
   - Famine doesn't automatically trigger plague (could add this)
   - Plague doesn't trigger coup (political opportunity)
   - Coup doesn't break alliances (maybe should?)
   - → V1.1 feature, not critical for V1.0

2. **Intervention AI Not Intelligent**
   - Nations will intervene but no complex strategy
   - Just military aid, no diplomatic pressure
   - → Fine for MVP, add smarts in V1.1

3. **Refugee System Incomplete**
   - Refugees move but don't integrate fully
   - Can cause temporary population spikes elsewhere
   - → Works for now, refine later

4. **Event Severity Not Adaptive**
   - All events have fixed impact regardless of nation size
   - Big empire + small plague = same effect
   - → Should scale by population/GDP, V1.1 improvement
