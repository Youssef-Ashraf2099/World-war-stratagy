## 🔧 TECHNICAL SPECIFICATIONS

### GameClock Model

```rust
pub struct GameClock {
    pub start_year: i32,          // Default: 2010
    pub start_month: u32,         // Default: 1
    pub start_day: u32,           // Default: 1
    pub hours_per_tick: u32,      // Default: 4
    pub speed: GameSpeed,         // Default: Normal
}

pub enum GameSpeed {
    Paused,      // 0 ticks per step
    Slow,        // 1 tick per step
    Normal,      // 3 ticks per step
    Fast,        // 7 ticks per step
    VeryFast,    // 15 ticks per step
}
```

### Datetime Calculation

```
elapsed_hours = tick * hours_per_tick
elapsed_days = elapsed_hours / 24
current_hour = (elapsed_hours % 24) as u32

date = start_date + Days(elapsed_days)
current_minute = 0  // Always on the hour for simplicity

formatted = format!("{:02}:{:02}, {} {}, {}", 
    hour, minute, day, month_name, year)

Example: "04:00, 1 January, 2010" at tick 1
```

### API Endpoints

**GET /world/clock**
```json
{
  "tick": 1000,
  "date_time": "04:00, 18 February, 2010",
  "game_clock": {
    "start_year": 2010,
    "start_month": 1,
    "start_day": 1,
    "hours_per_tick": 4,
    "speed": "normal",
    "speed_ticks_per_step": 3
  }
}
```

**POST /world/clock**
```json
{
  "start_year": 2015,           // Optional: change start year
  "start_month": 6,             // Optional
  "start_day": 10,              // Optional
  "hours_per_tick": 6,          // Optional: 6 hours per tick
  "speed": "fast"               // Optional: "paused", "slow", "normal", "fast", "very_fast"
}
```
