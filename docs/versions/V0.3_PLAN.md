**Phase Order Rationale:**

- AI decides before actions taken

- Combat after logistics (supply state affects strength)
- Occupation after combat (capture provinces)
- Stability after occupation (war weariness)
- Demographics last (apply casualties)

## 🎖️ **1. TACTICAL COMBAT SYSTEM**

### **1.1 Army System**

```rust
pub struct Army {
    pub army_id: ArmyId,
    pub owner: NationId,
    pub location: ProvinceId,

    // Unit composition
    pub infantry: u64,
    pub armor: u64,
    pub artillery: u64,

    // Combat state
    pub morale: f64,           // 0-100
    pub organization: f64,     // 0-100 (combat readiness)
    pub supply_state: f64,     // 0-1 (from LogisticsPhase)
    pub entrenchment: f64,     // 0-1 (defensive bonus)

    // Movement
    pub movement_points: f64,
    pub destination: Option<ProvinceId>,
}
```

**Combat Strength Formula:**

```
strength = (infantry * 1.0 + armor * 3.0 + artillery * 2.0)
         * (morale / 100.0)
         * (organization / 100.0)
         * supply_state
         * (1.0 + entrenchment * 0.5)
```

### **1.2 Province Battle Resolution**

```rust
pub struct ProvinceBattle {
    pub battle_id: BattleId,
    pub province_id: ProvinceId,
    pub tick_started: Tick,
    pub duration: u32,

    pub attackers: Vec<ArmyId>,
    pub defenders: Vec<ArmyId>,

    pub terrain_modifier: f64,
    pub weather_modifier: f64,
}
```

**Resolution Logic:**

1. Calculate total strength for each side
2. Apply terrain/weather modifiers
3. Calculate casualties (Lanchester's Square Law)
4. Apply casualties to armies
5. Check victory conditions:
   - Defender routed (< 30% strength) → Attacker victory
   - Attacker routed (< 30% strength) → Defender victory
   - Duration > 20 ticks → Stalemate

**Effects:**

- Cancel trade agreements
- Activate blockades

### **2.2 Occupation System**

```rust
pub struct OccupiedProvince {
    pub province_id: ProvinceId,
    pub occupier: NationId,
    pub original_owner: NationId,
    pub resistance: f64,  // 0-1, grows over time
}
```

**Effects:**

- Reduced production: `output *= (1.0 - resistance * 0.5)`
- Garrison cost: troops tied down
- Resistance grows: `+0.01/tick`
- High resistance causes casualties

### **3.2 Decision Tree**

```rust
impl BasicAI {
    pub fn decide_action(&self, world: &World) -> AIAction {
        let context = self.assess_situation(world);

        match self.personality {
            AIPersonality::Aggressive => {
                if context.military_advantage > 1.5 {
                    AIAction::DeclareWar { target }
                } else {
                    AIAction::BuildArmy
                }
            },
            AIPersonality::Defensive => {
                if context.being_attacked {
                    AIAction::DefendBorders
                } else {
                    AIAction::FocusEconomy
                }
            },
            AIPersonality::Balanced => {
                if context.military_advantage > 2.0 {
                    AIAction::DeclareWar { target }
                } else if context.legitimacy < 40.0 {
                    AIAction::FocusEconomy
                } else {
                    AIAction::BuildArmy
                }
            },
        }
    }
}
```

## 🤝 **4. NON-COMBAT EXPANSION**

### **4.1 Vassalization**

```rust
pub struct VassalOffer {
    pub from: NationId,
    pub to: NationId,
    pub terms: VassalTerms,
}

pub struct VassalTerms {
    pub tribute_percentage: f64,  // % of GDP
    pub military_access: bool,
    pub autonomy_level: AutonomyLevel,
}

pub enum AutonomyLevel {
    FullVassal,      // Almost annexed
    Protectorate,    // Defensive only
    Tributary,       // Just tribute
}
```

**Acceptance Logic:**

- Accept if power ratio > 5:1
- Accept if under attack AND power ratio > 2:1
- Accept if generous terms (< 10% tribute, protectorate)

**Effects:**

- Tribute transferred each tick
- Loyalty system (0-100)
- Low loyalty → independence war

### **4.2 Annexation**

```rust
pub struct AnnexationOffer {
    pub annexer: NationId,
    pub target: NationId,
    pub compensation: f64,
}
```

**Acceptance:**

- Only if legitimacy < 15 AND compensation > GDP \* 5
- Very rare

**Execution:**

- Transfer all provinces
- Merge populations
- Despawn target nation
